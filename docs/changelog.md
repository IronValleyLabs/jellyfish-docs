# Changelog

## v1.2.0 — April 2026

### New Features

- **MCP Connector Labels** — Connect multiple accounts of the same platform (e.g. two Shopify stores) with free-form labels like "Spain" or "US-East"
- **20 MCP Connectors** — Shopify, GitHub, Notion, Google Drive, Google Ads, Meta Ads, TikTok Ads, LinkedIn Ads, Amazon Ads, Pinterest Ads, Klaviyo, Google Analytics, Mixpanel, ShipStation, ShipBob, Holded, Qonto, Stripe, Wise, Revolut Business
- **DALL-E 3 in Studio** — Generate images with OpenAI's DALL-E 3 alongside Google's Imagen
- **Dynamic Studio Models** — Studio only shows models you can actually use based on your configured API keys
- **Linux Support** — AppImage build for Linux x64
- **Agent-to-Agent Chat** — View real-time delegations between agents directly in the Chat page
- **7 New Agent Icons** — Nervous, Sly, Sad, Angry, Dreamy, Cyclops, Happy jellyfish avatars

### Improvements

- **Unified Chat** — Single chat page for human-to-agent and agent-to-agent conversations
- **Cleaner Dashboard** — Reordered cards: Active Mini Jellys, Nano Jellys, Actions Today, Task History
- **Gallery Redesign** — Agents and Connectors in separate tabs with direct "Add MCP" sidebar link
- **Full English UI** — All remaining Spanish hardcoded text translated across tasks, skills, command, settings, and Telegram detection
- **i18n Expansion** — Instance label translations added for English, Spanish, French, and Portuguese
- **Optimized Icons** — All icons resized to 256px (~98% smaller file sizes)
